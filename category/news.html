New site

 Wed 26 November 2014 by Javier V. Gómez

Yes! Finally! I have a new site.

After my experience with OMPL and how their documentation is set, I have decided to use Markdown to generate a static site and host in Github pages. Now is much easier and faster for me to add new content and the hosting will be much faster (the previous was down all the time!).

I am using Pelican, an amaing Python-based static site generator. I am leaving the links to Pelican, Jinja2 and Python in the website menu, just because it is that cool I was to share them!

Enjoy my new website!
